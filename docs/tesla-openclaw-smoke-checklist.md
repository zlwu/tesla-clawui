# Tesla OpenClaw MVP 冒烟清单

## 环境准备

1. 项目根存在 `.env`
2. 执行过 `npm install`
3. 已启动 `npm run dev:server`
4. 已启动 `npm run dev:web`

## 冒烟步骤

1. 打开页面，确认会话自动建立
2. 输入一条文本，确认收到 assistant 回复
3. 录一段语音，确认看到 transcript 和 assistant 回复
4. 连续交互 5 轮，确认页面仍清晰可用
5. 刷新页面，确认最近消息仍能恢复
6. 断开网络后再发送一次，确认页面提示失败且可点击“重试上一步”
7. 恢复网络，点击“重试上一步”，确认流程继续可用
8. 对同一个 `requestId` 重复提交，确认不会产生重复消息

## 记录项

- 日期和时间
- Tesla 软件版本或浏览器现象
- 测试项是否通过
- 错误码或报错文案
- 是否需要继续优化
